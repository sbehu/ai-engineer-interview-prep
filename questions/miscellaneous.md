GenAI / Agentic AI Interview Questions
A curated list of interview questions covering RAG, Agentic AI, LLMs & Prompting, and Python/SQL coding challenges.

🤖 GenAI & RAG

Walk me through your RAG pipeline — from ingestion to retrieval to generation.
How do you handle chunking? What chunk size works best?
Why ChromaDB? When would you choose Pinecone over it?
How do you evaluate RAG performance? (RAGAS, human eval, etc.)
What are the failure modes of RAG? (Bad retrieval → bad generation, hallucination)


🧠 Agentic AI

What is an AI Agent? How is it different from a simple LLM call?
How do you design a multi-agent system?
How do you handle tool calling and decision-making?
What frameworks have you used? (LangChain, LangGraph, etc.)
How do you evaluate an agent's performance?


💬 LLMs & Prompting

What's the difference between system prompt and user prompt?
How do you handle conversation memory with stateless LLMs?
What is chain-of-thought prompting? When do you use it?
Why temperature = 0.3 for code generation?
How do you prevent prompt injection?


🐍 Python & SQL Coding

Write a function to check if a list contains duplicates.
Write a function to check if two strings are anagrams.
Write a SQL query to find duplicate rows in a table.
Implement a simple RAG retrieval function.
